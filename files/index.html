Source Files

About

This page lists source files for almost all releases by Ronyx69.

The files are provided as they are - some of them might contain 3ds Max or Maya scene files, Photoshop documents, references, notes, scripts, various intermediary files such as bump maps used for creating normal maps, or scale guidelines. Some might not contain the final files used for importing into the game, and the importing process might not be straight forward, as it may require mods or scripts.

The files may be used for learning purposes, or for creating your own assets or versions of the already released assets. You don't need to ask for permission to use these files. Credit and a link to this page is appreciated but not required.

Do not use the Ronyx69 logo, branding, or thumbnail design style in a misleading way, for example - making it seem like Ronyx was personally involved in your projects.

Files

Each item is displayed with the steam workshop item id and title, and links to a file on google drive. Some items, usually packs, are grouped together and link to a single file.
